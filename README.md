# Machine Learning Projects

This repository contains a collection of Machine Learning projects implemented using Python and Scikit-Learn. The projects cover data preprocessing, exploratory data analysis (EDA), model training, and evaluation techniques.

## Projects

* Iris Flower Classification
* Heart Disease Prediction
* House Price Prediction
* Insurance Cost Prediction
* Loan Approval Prediction
* Employee Turnover Prediction
* K-Nearest Neighbors (KNN)
* Naive Bayes Classification

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Machine Learning Concepts

* Data Cleaning & Preprocessing
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Regression Models
* Classification Models
* Model Evaluation
